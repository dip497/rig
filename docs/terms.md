## Name, logo, and brand

*"Rig"* and the Rig logo are project identifiers. We prefer they be
used:

- **Allowed without asking:** linking to the project, writing
  tutorials, announcing a Rig bundle you created, writing a post that
  references Rig by name, using the word "Rig" in documentation or
  slide decks.
- **Please ask first:** hosting a service called "Rig" or using the
  Rig logo in the name of a company, product, or paid offering that
  might be confused with the project itself.

We are keeping this section intentionally light and pre-legal. If the
project grows to need a formal trademark policy, we will adopt one and
this document will be updated with notice.
